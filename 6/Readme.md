# Упаковываем приложение VueJS в Docker контейнер

Это тестовое приложение, которое мы написали чтобы позднакогмится с базовыми навыками создания, сборки и развертывания приложения на jueJS 

## Сначала собираем
```
docker build --target development -t app-dev .
```

## Потом запускаем
```
docker run --rm -p 5173:5173 app-dev
```

[!WARNING]
Обратите внимание, что
Vite не поддерживает опцию --root через командную строку напрямую для команды vite build. 
Он ожидает, что вы укажете корень проекта в конфигурационном файле vite.config.js

## Теперь проверим нашу итоговую сборку
```
docker build -t my-nginx-app .
```

## Проверим работу
```
docker run -d -p 8080:80 --name react-app-container my-nginx-app
```

## Так как мы прокидываем порт смотрим тут
http://localhost:8080
